# Cost Controls Page

**Route:** `/cost`
**Required Role:** Admin
**Component:** `CostControlsPage.tsx`

Monitor LLM token usage, configure per-tenant cost budgets, and check budget enforcement status.

## Tabs

### Token Usage Tab

**Period Selector**: 7, 14, 30, or 90 days.

**Summary Cards**:
- Total Tokens consumed
- Total Requests processed
- Cache Hit Rate (embedding cache)
- Estimated Cost (USD)

**Budget Utilization Cards**: Daily and Monthly budget usage percentage.

**Daily Breakdown Table**:

| Column | Description |
|--------|-------------|
| Date | Day of activity |
| Requests | Number of chat requests |
| Prompt Tokens | Tokens sent to the model |
| Completion Tokens | Tokens generated by the model |
| Embedding Tokens | Tokens used for embeddings |
| Cache Hits | Embedding cache hits |
| Est. Cost | Estimated cost in USD |

### Settings Tab

Per-tenant cost optimization settings with nullable overrides (null fields fall back to global defaults).

| Setting | Description |
|---------|-------------|
| Daily Token Budget | Maximum tokens per day |
| Monthly Token Budget | Maximum tokens per month |
| Max Prompt Tokens/Query | Per-query prompt token cap |
| Max Evidence Chunks | Maximum chunks in prompt |
| Budget Alert Threshold | Percentage at which to warn |
| Embedding Cache Enabled | Toggle cache for embeddings |
| Embedding Cache TTL | Cache time-to-live in hours |
| Retrieval Compression | Truncate long evidence chunks |

Actions: Edit (toggle edit mode), Reset to Defaults (removes tenant overrides).

### Budget Status Tab

Real-time budget enforcement check:
- **Status**: Allowed or Denied (color-coded)
- **Daily Utilization**: percentage of daily budget used
- **Monthly Utilization**: percentage of monthly budget used
- **Warning**: shown when alert threshold exceeded
- **Denial Reason**: shown when budget exhausted

## API Endpoints Used

- `GET /api/admin/token-usage/summary` — usage summary for period
- `GET /api/admin/token-usage/daily` — daily breakdown
- `GET /api/admin/cost-settings` — current settings
- `PUT /api/admin/cost-settings` — update settings
- `DELETE /api/admin/cost-settings` — reset to defaults
- `GET /api/admin/token-usage/budget-check` — budget status
